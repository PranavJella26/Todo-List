# Todo-List
## Todo List Application
A simple Todo List built using vanilla JavaScript that demonstrates clean state management, DOM manipulation, and local storage persistence.

## Features
Add, delete, and toggle tasks, 
Unique task identification, 
Single Source of Truth using a state array, 
Persistent data with Local Storage, 
Dynamic DOM rendering, 

## Core Concepts
Todo Object: { text, uniqueNo, isChecked },
State Array: Centralized todo list data,
Local Storage: Saves tasks across sessions,
Event Handling: Syncs UI with data updates,

## Tech Stack
HTML,
CSS,
JavaScript (ES6),

## Outcome
A fully functional Todo List showcasing fundamental web application principles using pure JavaScript.
